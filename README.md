# 八尾路智ケアシステム

## 概要
システム目的
介護施設内の消耗品（おむつ・衛生用品等）と介護資産（車椅子・介護機器等の備品）について、在庫数量・所在・払出履歴を一元管理するシステム。
現状は紙・手書き台帳による管理のため、棚卸時の手作業による照合に多大な工数がかかっており、
誰がいつ何を使ったかの追跡も困難な状態にある。また発注点管理が行われておらず、
消耗品の欠品がケアの質に影響するリスクも存在する。本システムの導入により、
棚卸・入出庫業務の手戻りと欠品リスクを低減し、現場スタッフの業務効率化と施設運営のコスト可視化を実現する。
